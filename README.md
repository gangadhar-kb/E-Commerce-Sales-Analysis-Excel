# E-Commerce Sales Performance Dashboard (Excel)

## 📊 Project Overview
An interactive E-Commerce Performance Dashboard built from scratch using Microsoft Excel. This project demonstrates end-to-end data processing, from ETL (Extract, Transform, Load) to building a dynamic reporting layout for business stakeholders.

## 🛠️ Tech Stack & Skills
* **Data Cleaning & ETL:** Power Query (Removing duplicates, handling blanks, date formatting)
* **Data Modeling:** Calculated Fields, M-Code Custom Columns (Profit, Profit Margin %)
* **Analysis:** Pivot Tables (Aggregation, Top 10 Product Filtering, Sorting)
* **Visualization:** Advanced Charting, Dynamic Slicers, Report Connections, UI Layout Design

## 📈 Key Insights Generated
* **Top Revenue Drivers:** Identified the top 10 products contributing to total sales.
* **Regional Distribution:** Tracked sales performance across Central, East, South, and West regions.
* **Monthly Trends:** Visualized sales trajectories over the year to pinpoint peak demand seasons.
